## 📌 Problem Statement

A mid-sized financial company, SecureTrust Bank, provides personal and home loans to customers across India. Currently, loan approvals are handled through a manual verification process where loan officers evaluate applications based on income, employment details, credit history, and other factors.

This manual approach has several limitations:
- It is time-consuming
- It can be biased and inconsistent
- Good customers may get rejected
- High-risk customers may get approved

To address these challenges, the goal of this project is to build an **intelligent loan approval system using Machine Learning** that can automatically analyze applicant data and predict whether a loan should be **Approved or Rejected** before final human verification.

---

## 🎯 Objective
Develop a machine learning model that:
- Learns patterns from historical loan data
- Predicts loan approval status accurately
- Helps reduce risk and improve decision-making speed

---

## 📊 Dataset Description
Each record represents a loan applicant and includes:

- Applicant Income & Co-applicant Income  
- Employment Status  
- Age, Marital Status, Dependents  
- Credit Score & Existing Loans  
- Debt-to-Income Ratio (DTI)  
- Savings & Collateral Value  
- Loan Amount & Loan Term  
- Loan Purpose & Property Area  
- Education Level & Gender  
- Employer Category  

**Target Variable:**
- `Loan_Approved` → 1 (Approved), 0 (Rejected)
